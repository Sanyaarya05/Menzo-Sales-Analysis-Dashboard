# 📊 Menzo Sales Analysis Dashboard

An end-to-end data analytics project analyzing retail sales data to uncover business insights and support decision-making.

---

## 🚀 Project Overview

This project focuses on analyzing sales data from a menswear retail store to understand:

- Sales performance
- Customer behavior
- Product demand
- Pricing strategy

---

## 🛠️ Tools & Technologies

- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI

---

## 📈 Key Insights

- Topwear contributes the highest revenue
- High price range products generate maximum sales
- Cash is the most preferred payment method
- Average Order Value is ~₹2000

---

## 📊 Dashboard Features

- KPI tracking (Revenue, Profit, Orders, AOV)
- Monthly sales trend analysis
- Category-wise performance
- Product-level insights
- Payment mode analysis

---


## 📁 Dataset

The dataset contains ~2000 retail transactions including:
- Product details
- Pricing
- Quantity
- Payment mode
- Revenue & profit
